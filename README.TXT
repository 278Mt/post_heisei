仕様書

ご利用ありがとうございます。以下が仕様書です。

初期設定
1. 「combo.csv」に出品者IDを保存しておく（いちばん上は必ず「項目」のままにしておいてください）。
2. フォルダーのなかにある「chromedriver」というファイルを「C:」直下にドラッグアンドドロップで移動させる。

以上で初期設定は終了です。

実行
0. tool.exeを起動する
1. 「出品者ID」を選択する。
2. 「ワードによる絞り込みをする」でワードを入力する(しなくても良い)
3. 「CSVの出力ファイル」を入力する(デフォルトは「デスクトップ/output.csv」)
4. 「データ取得」をクリックする。
5. 「選択して実行」をクリックする。
6. 終了するまで、コマンドプロンプトにログが表示されるので確認する。
7. 終了後、ファイルが自動で開くので確認する。
